# ✈️ Travel Destinations Website

A beginner-friendly multi-page travel website built with **HTML** and **CSS** — no frameworks, no JavaScript libraries, just pure web basics.

## 🌍 Live Pages

| Page | Destination |
|------|------------|
| `index.html` | Home — all destination cards |
| `paris.html` | Paris, France |
| `tokyo.html` | Tokyo, Japan |
| `santorini.html` | Santorini, Greece |
| `sydney.html` | Sydney, Australia |
| `maldives.html` | Maldives |
| `newyork.html` | New York, USA |

## ✨ Features

- 🃏 **Responsive card grid** — cards wrap neatly on all screen sizes
- 🖱️ **Hover transitions** — cards lift, images zoom, buttons scale smoothly
- 🔗 **Multi-page navigation** — Explore buttons link to individual destination pages
- 📸 **Photo galleries** — three images per destination with hover zoom effect
- 📊 **Quick facts** — language, currency, climate, and rating per destination
- 📍 **Top places to visit** — listed with animated hover effect
- 🗓️ **Best time to visit** — travel tips for each destination
- ← **Back button** — returns to home page from any destination

## 🛠️ Tech Used

- HTML5
- CSS3 (Flexbox, Grid, Transitions, Transforms, Media Queries)
- Google Fonts — Poppins
- Unsplash (free images via URL)

## 🚀 How to Run

1. Download or clone this repository
2. Keep all `.html` files in the **same folder**
3. Open `index.html` in any browser
4. Click **Explore →** on any card to visit that destination's page

```bash
git clone https://github.com/your-username/travel-destinations.git
cd travel-destinations
open index.html
```
